# Book Management System

## 🔗 Live Demo
https://krut.runasp.net

## 📌 Description
This is a web-based Book Management System developed using ASP.NET and SQL Server. It allows users to manage books, authors, and categories.

## 🚀 Features
- Add, Update, Delete Books
- Manage Authors and Categories
- User-friendly interface

## 🛠 Tech Stack
- ASP.NET (C#)
- SQL Server
- HTML, CSS
